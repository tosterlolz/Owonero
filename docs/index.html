Wallet Stats

Enter a wallet public key (hex) to view its current reported hashrate and on-chain balance. The page will poll the node periodically. Note: the daemon must expose an HTTP API (e.g. /api/wallethashrate?addr= and /api/walletbalance?addr=) for live data; otherwise this UI will show a connection error.
